# ⚡ Aegis — AI Financial Stress Tester

> Know your risks before it's too late.

**Live Demo:** https://aegis-fire.vercel.app

## About

Aegis is an AI-powered Post-FIRE (Financial Independence, Retire Early) stress-testing tool for Indonesian investors. It simulates whether your retirement fund will survive realistic macroeconomic crisis scenarios generated by AI.

Users input their portfolio profile — asset allocation (stocks, bonds, cash, crypto), total retirement fund, monthly expenses, and retirement age. Aegis sends this data to an AI model with strict prompt engineering that generates three crisis scenarios specific to that portfolio in structured JSON format, then visualizes the results as an interactive dashboard.

## How It Works

1. User inputs portfolio profile and financial targets
2. Data is sent to `/api/analyze`
3. API route sends prompt to Gemini via Google AI SDK
4. AI generates 3 realistic crisis scenarios as structured JSON
5. Frontend renders results as an interactive dashboard with area charts, summary metrics, and actionable recommendations

## Features

- **AI-Powered Scenarios** — 3 unique crisis scenarios per analysis, specific to your portfolio
- **Post-FIRE & Already Retired modes** — different simulation context for each
- **Interactive Charts** — 30-year fund trajectory visualization per scenario
- **Alternative Solution** — if fund depletes, AI calculates minimum fund and defensive allocation needed
- **Safe Withdrawal Rate** — personalized SWR recommendation with practical Rupiah breakdown
- **Simulation Disclaimer** — clear educational framing for all results

## Tech Stack

- **Framework:** Next.js 16.2 (App Router)
- **Styling:** Tailwind CSS v4.1
- **Charts:** Recharts 3.8
- **AI:** Google Gemini via Google AI SDK (with multi-model fallback)
- **Deploy:** Vercel
